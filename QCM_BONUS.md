## COMPÉTENCES & FORMATIONS

**Q1.** Quelle méthode d'évaluation des compétences est la plus objective ?
- A) Auto-évaluation
- B) Tests techniques
- C) Évaluation par les pairs X
- D) Entretien individuel

**Q2.** Quel est l'inconvénient principal de l'auto-évaluation ?
- A) C'est trop long
- B) Dépend de l'honnêteté et de la confiance en soi X
- C) C'est trop cher
- D) Nécessite un formateur

**Q3.** Les formations externes peuvent être financées via :
- A) Le CPF uniquement
- B) L'OPCO X
- C) L'entreprise uniquement
- D) Aucun financement possible

**Q4.** Une formation interne est animée par :
- A) Un organisme externe
- B) Un consultant
- C) Un senior ou Tech Lead de l'entreprise X
- D) Un auditeur

## ROADMAP

**Q5.** Quelle est la différence entre Vision et Mission d'entreprise ?
- A) Vision = moyens, Mission = futur idéal
- B) Vision = futur idéal, Mission = moyens X
- C) C'est la même chose
- D) Vision = court terme, Mission = long terme

**Q6.** Quels sont les 5 critères d'une bonne Vision d'entreprise ?
- A) SMART
- B) INVEST
- C) Imaginable, Désirable, Concentrée, Flexible, Facile à expliquer X
- D) Mesurable, Atteignable, Réaliste, Temporel

**Q7.** La Vision produit doit contenir :
- A) Uniquement le nom du produit
- B) Produit + Public + Problème + Solution + Avantages X
- C) Uniquement les fonctionnalités
- D) Le budget et les délais

## OKR & COÛTS

**Q8.** Que signifie OKR ?
- A) Objective Key Results
- B) Objectives and Key Results X
- C) Organization Key Resources
- D) Optimal Key Roadmap

**Q9.** La règle du 3-3-3 consiste à :
- A) Avoir 3 équipes de 3 personnes
- B) Planifier sur 3 trimestres sans dates précises X
- C) Faire 3 sprints de 3 semaines
- D) Avoir 3 OKR par équipe

**Q10.** Quel pourcentage de charges patronales ajouter au coût brut RH ?
- A) 30%
- B) 45% X
- C) 50%
- D) 60%

---

# CORRIGÉ

| Q | Réponse | Explication |
|---|---------|-------------|
| Q1 | **C** | Évaluation par les pairs = plus objectif car regard externe |
| Q2 | **B** | Dépend de l'honnêteté et confiance en soi |
| Q3 | **B** | OPCO = Opérateur de Compétences (finance les formations) |
| Q4 | **C** | Interne = animée par un expert de l'entreprise |
| Q5 | **B** | Vision = futur idéal / Mission = moyens pour y arriver |
| Q6 | **C** | 5 critères : Imaginable, Désirable, Concentrée, Flexible, Facile |
| Q7 | **B** | Vision produit = Produit + Public + Problème + Solution + Avantages |
| Q8 | **B** | Objectives AND Key Results |
| Q9 | **B** | Planifier sur 3 trimestres, pas de dates précises |
| Q10 | **B** | 45% de charges patronales |
